 Database Setup

To run the project with full data:

1. Open **SQL Server Management Studio (SSMS)**
2. Right-click on **Databases**
3. Select **Restore Database**
4. Choose **Device**
5. Select the file: /Database/ShoppingData.bak
6. Complete the restore process
- Product images are loaded using **online URLs (Unsplash)**  
- No local image setup is required
